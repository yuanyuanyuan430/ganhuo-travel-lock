# Travel Guide Template

## Intake

Ask at most 3 blocking questions when missing:

- 出发地
- 天数/日期
- 人群类型: 亲子、毕业生、朋友、情侣、内容创作者

Use explicit assumptions for missing soft fields:

- 预算: 中等
- 节奏: 中等
- 交通: 不自驾
- 范围: 国内游
- 偏好: 热门景点和低踩坑路线兼顾

Useful optional fields:

- 目的地或候选城市
- 人数、孩子年龄、人均预算
- 偏好: 海边、避暑、乐园、博物馆、美食、拍照、自然、Citywalk
- 节奏: 轻松、中等、特种兵
- 自驾、怕热怕晒、低消费、小众替代

## Child Age Rules

- 3-6 岁: 少换酒店，少排队，午睡优先，半天一个主点，优先乐园、动物、亲水、室内馆。
- 7-12 岁: 可加入博物馆、自然观察、轻徒步、研学任务；每天保留一个自由玩耍时间。
- 13 岁以上: 可增加 Citywalk、夜景、拍照、美食探索和轻挑战路线，但仍要控制暑期暴晒。

## Summer Risk Checks

Always call out these risks when relevant:

- 高温暴晒: 午间换室内，准备补水、防晒、打车预算。
- 暴雨/台风: 海边、山区、海岛、江浙沪和华南路线必须查预警。
- 山洪/落石: 峡谷、漂流、徒步、溶洞周边要查景区公告。
- 限流预约: 博物馆、热门乐园、演出、夜游、名山大川要提前核验。
- 涨价: 暑期机酒波动大，只给预算构成，不编精确价。
- 排队: 热门乐园和网红餐厅要给早到、错峰、替代点。
- 交通紧张: 高铁票、机场接驳、末班车、跨城日行程必须核验。

## Xiaohongshu Angle

When the user asks for 小红书攻略, include:

- 标题方向: 人群 + 城市 + 天数 + 结果感，例如 "上海出发 3 天带娃避暑，不累也不踩坑"
- 封面信息: 城市、天数、预算、适合年龄或人群
- 正文结构: 先结论，再路线，再避坑，再核验清单
- 内容边界: 不承诺最低价、不伪造排队时间、不写未经核验的票价
