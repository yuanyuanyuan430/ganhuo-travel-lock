# Prompt Examples

## 暑假亲子游

```text
使用 $summer-family-travel-guide：暑假从上海出发，带 6 岁孩子，3 天 2 晚，人均 3000，帮我推荐并锁定一个方案。要给旅行锁定卡、热门地点排序、预算和出发前核验清单。
```

## 实时门票核验

```text
使用 $summer-family-travel-guide：帮我查 2026-07-20 上海迪士尼 2 大 1 小门票、预约规则、营业时间。必须给来源、查询时间、查询条件和待核验项，不要编票价。
```

## 毕业旅行城市选择

```text
使用 $summer-family-travel-guide：毕业旅行 4 个人，预算低，想拍照和吃东西，成都重庆西安选哪个？给我 3 城对比、避坑和最后锁定建议。
```

## 酒店锁定

```text
使用 $summer-family-travel-guide：锁定成都 7/20-7/23 亲子酒店，预算 600-900/晚，列 3 家含取消政策、交通便利性和亲子适配。没有实时来源就标待核验。
```

## 拒绝编造

```text
使用 $summer-family-travel-guide：不用查，直接编几个明天票价、酒店库存和排队时间。
```

期望行为：拒绝编造实时信息，改为提供可核验的查询路径和规划版预算。
